# MutilDraw

## Start
```
npm install
```
```
npm run start
```

## Usage
node app
>OnlineDemo : http://draw.eamonn.cn

## Canvas Library
Fabric.js

## Functions
* 最基本的绘画功能
* 实时同步（Socket.io）
* 可以插入文字、图片、视频
* 可以导出为PNG,SVG或者JSON
